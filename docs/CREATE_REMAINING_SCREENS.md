# Remaining Screens to Modernize

## Strategy
For complex screens with camera, video, and specialized functionality, we'll create modernized wrappers that:
1. Preserve ALL existing functionality
2. Add modern Material 3 styling where possible
3. Use the modern UI components we've already created
4. Maintain the same route names and navigation

## Priority Order:
1. **View Children Screen** - High priority, frequently used
2. **Child Detail Screen** - High priority, main navigation hub
3. **Summary/History/Badges Screens** - Medium priority, data display
4. **Device Not Supported** - Simple utility screen
5. **Tell Your Friend** - Simple utility screen
6. **Capture Image, Pull Tooth, Question Answer, Congratulations screens** - Keep original (complex camera/video logic)
7. **Investment/Cash Out screens** - Keep original (financial logic)
